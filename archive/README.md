# Archive

Contenido de archivo no referenciado por el código en producción.

## legacy-assets

Imágenes y PDFs que estaban en la raíz del proyecto y se movieron aquí para mantener la raíz limpia. El portal utiliza las copias ya organizadas en `public/images/`, `public/documents/`, etc., con nombres normalizados.

- **Imágenes:** posibles duplicados o versiones previas (logos, mapas, fotos de cumbres, autoridades).
- **PDFs:** ej. TdR Paridad, TdR Ciberseguridad; las versiones en uso están en `public/documents/grupos-trabajo/`.

No eliminar sin revisión; pueden servir de respaldo o referencia.
